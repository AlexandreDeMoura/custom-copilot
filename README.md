The goal of this project is to help you customize your workflow using OpenAI and Anthropic latest models.

## Features

- Chat with OpenAI and Anthropic latest models by using your own API keys
- Use presets or define your own prompts
- Save your favorite presets
- Save your favorite prompts and add categories to navigate through them easily
- French and English supported
- Dark and light themes

## How to use

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Use your OpenAI and Anthropic API keys
5. Enjoy !
